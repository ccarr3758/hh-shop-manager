# Tech Account All Jobs + Leaderboard Fix V6

Fixes the technician login view where jobs and leaderboard stats were only calculating from the signed-in technician's filtered job list.

## Change
- Technician accounts still cannot edit existing jobs unless their role allows it.
- Technician accounts can now see the full shop job set for dashboards, jobs, and leaderboards.
- Leaderboard stats now populate for all technicians while logged into a tech account.

Deploy over V5.
