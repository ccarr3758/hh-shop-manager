# H&H Shop Manager v3.0.1 Manager View UI Update

Scope intentionally limited to Manager View UI only.

Changes included:
- Compact single-row bottom navigation.
- Larger technician name inside each mobile job card.
- Larger, more prominent Book Time card.
- Live Time Remaining that changes to Over by X min after book time.
- Thin progress bar based on actual start time.

No scheduling, helper, Supabase, or performance formula changes were made for this update.
