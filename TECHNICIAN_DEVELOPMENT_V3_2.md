# v3.2 Technician Development

Added inside the existing Performance page.

Visible only to:
- Admin
- Manager
- Foreman

Not visible to technicians or service writers.

Included metrics:
- Efficiency trend
- Book hours produced
- Jobs completed
- Jobs over book time
- Average minutes over book
- Helper hours given
- Jobs assisted
- Job mix / specialties
- Automated strengths
- Automated focus areas
- Automated coaching recommendations

Explicitly excluded:
- Clock-in / clock-out metrics
- Scheduling metrics
- Time between jobs
- QC failures
- Customer complaints
- Customer compliments

Also updated the top-right Performance KPI cards so the values are much larger and easier to read.
