# Helper Hours Breakdown Update

This update exposes helper labor as a visible performance metric instead of only using it behind the scenes.

## Added to technician performance cards

Each technician now shows:

- Primary Book Hours
- Helper Book Hours
- Total Book Hours
- Actual Hours
- Hours Helped
- Help Received
- Jobs Completed

## Rules

- Helper hours add to book hours and actual hours for efficiency.
- Helper hours do not add to job count.
- Hours Helped = time this technician spent assisting other lead technicians.
- Help Received = helper time assigned to jobs where this technician was the lead.
- Primary Book Hours stay separate so a technician's own job output is still visible.

## Dashboard

The main dashboard now exposes helper book hours and helper actual hours as shop-level KPIs.
